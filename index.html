Правила начисления и использования баллов

  • Начисление баллов производится ежемесячно, в период с 1 по 3 число месяца.
  • Баллы можно копить без ограничений по срокам.
  • Проверка баланса осуществляется в Google Таблице «ЭтажиPlay» (лист Балансы).
  • Сгорание баллов: если партнер прекращает сотрудничество с компанией, все его накопленные баллы аннулируются.
  • Командные активности: списание баллов с нескольких партнеров на один приз возможно только после согласования с директором.

Методика расчета рейтинга

Рейтинг формируется отдельно для партнеров каждой из трех групп стажа: 0–3, 4–6 и 6+ месяцев.

Правила расчета рейтинга и определения позиции одинаковы для всех групп стажа:

  • -В рейтинге выше тот партнер, который заработал больше баллов по итогам расчета
  • -Если несколько партнеров заработали одинаковую сумму, то их порядок в рейтинге определяется исходя из % выполнения опережающих показателей
    (см. таблицу ЭтажиPlay)

!!! Все показатели рассчитываются за текущий календарный месяц !!!

Сумма начисления баллов по итогам месяца

Как считать сумму начисления в зависимости от наличия задатка и выполнения опережающих показателей?

Условие % выполнения опережающих показателей Сумма баллов к начислению
Есть задаток до 80% 500
Есть задаток от 80% до 100% 1 000
Есть задаток более 100% 1 500
Нет задатка до 80% 0
Нет задатка от 80% до 100% 200
Нет задатка более 100% 500

Баллы начисляются по итогам месяца согласно наличию задатка и проценту выполнения опережающих показателей.
Максимальное значение % выполнения — 120%.

% выполнения опережающих показателей определяется как отношение фактического значения к плановому

% выполнения опережающих показателей Не может превышать 120% — даже если фактические результаты намного лучше плана, максимальное значение, которое учитывается, — это 120%. Всё, что выше, не засчитывается.
В общем зачете и по каждому отдельному показателю — ограничение действует как на общий итоговый процент, так и на каждый отдельный показатель.
